## 0. הרוח — במשפט אחד

**נגן מוזיקה שמנהל כסף.** כהה, זכוכיתי, זוהר מבפנים, גאומטרי-מודרני.
ההשראה: Bankly (גרדיאנטים+זכוכית), אפליקציות mood (כדורי זוהר), Linear (חדות).
לא "דשבורד SaaS". לא שטוח. **עומק, גרדיאנט, נשמה בכל כרטיס.**

---

## 1. טיפוגרפיה

```
כותרות + מספרים:  'Space Grotesk'  (גאומטרי-מודרני, כמו Bankly)
                  משקלים: 400/500/600/700
גוף עברי:         'Heebo'          משקלים: 400/500/700/800
```

טען מ-Google Fonts:
```
https://fonts.googleapis.com/css2?family=Space+Grotesk:wght@400;500;600;700&family=Heebo:wght@400;500;700;800&display=swap
```

חוקים:
- **כל מספר** (כסף, ימים, ספירות, תאריכים) → Space Grotesk, `letter-spacing:-0.02em`, `direction:ltr`
- כותרות עברית → Heebo 700/800
- טקסט גוף עברית → Heebo 400/500
- תוויות/labels → Heebo 600/700, גודל 12-13px
- **אין** font-family ברירת מחדל של המערכת. שני הפונטים בלבד.

## 2. פלטה

```css
/* רקעים — כהה עמוק, לא שחור טהור */
--bg-base:     #080611;   /* רקע האפליקציה */
--bg-panel:    #0F0D1C;   /* פאנלים גדולים */

/* אקסנטים — חשמליים */
--violet:      #8B5CF6;   /* סגול ראשי */
--violet-lt:   #C084FC;   /* סגול בהיר (טקסט/זוהר) */
--violet-dk:   #6D28D9;   /* סגול כהה (גרדיאנט) */
--cyan:        #38BDF8;   /* תכלת — מידע/התחייבות */
--cyan-dk:     #0284C7;
--rose:        #FB7185;   /* אדום-ורוד — קריטי */
--rose-dk:     #E11D48;

/* מד VU */
--vu-green:    #4ADE80;
--vu-amber:    #FCD34D;
--vu-red:      #FB7185;

/* טקסט */
--ink:         #FFFFFF;
--ink-soft:    #B4B9D0;
--ink-dim:     #8B8FA8;
--ink-faint:   #64748B;
```

### חלוקת צבעים סמנטית — קריטי
- **סגול** = אקסנט ראשי, ניווט, פעולות רגילות
- **תכלת** = מידע, התחייבות פתוחה (**לא** חוב!)
- **אדום-ורוד** = קריטי, חוב באיחור, התראות דחופות
- **ירוק** = תקין, שולם, בזמן
- **ענבר** = אזהרה

חוב והתחייבות הם **צבעים שונים** כי הם דברים שונים.

## 4. כרטיסים — זכוכית (glassmorphism)

### כרטיס ראשי (מספרים גדולים, רדאר)
```css
background: linear-gradient(135deg, rgba(139,92,246,0.22), rgba(30,20,55,0.4));
backdrop-filter: blur(20px);
border: 1px solid rgba(192,132,252,0.3);
border-radius: 22px;
padding: 22px 24px;
box-shadow: 0 8px 32px rgba(139,92,246,0.2), inset 0 1px 0 rgba(255,255,255,0.1);
position: relative;
overflow: hidden;
```

**כדור זוהר בפינה** (החתימה מאפליקציות ה-mood) — בתוך כל כרטיס ראשי:
```css
/* אלמנט מוחלט בפינה, מתחת לתוכן */
position: absolute; top: -30px; left: -30px;
width: 120px; height: 120px;
background: radial-gradient(circle, rgba(192,132,252,0.4), transparent 70%);
border-radius: 50%; filter: blur(20px);
```
הצבע תואם לסוג הכרטיס: סגול/תכלת/ורוד.

### כרטיס משני (מודולים קטנים)
```css
background: linear-gradient(135deg, rgba(255,255,255,0.06), rgba(255,255,255,0.02));
backdrop-filter: blur(12px);
border: 1px solid rgba(255,255,255,0.1);
border-radius: 18px;
padding: 16px 18px;
```
כרטיס עם 0 / לא-פעיל → `opacity: 0.5`, בלי גרדיאנט צבעוני.

## 5. אנימציות — float-up בהובר (החתימה)

**זה האפקט המרכזי.** ב-hover על כרטיס:
- הכרטיס עולה: `translateY(-4px)`
- **התווית שבתוכו עולה בנוסף**: `translateY(-3px)` (float-up)
- הצל מתעצם, הגבול מואר

```css
.card { transition: transform .3s cubic-bezier(.2,.8,.2,1), box-shadow .3s, border-color .3s; }
.card .label { transition: transform .3s cubic-bezier(.2,.8,.2,1), color .3s; }
.card:hover { transform: translateY(-4px); }
.card:hover .label { transform: translateY(-3px); }
```

כרטיס משני: `translateY(-5px)` + הגבול מקבל את צבע האקסנט + צל צבעוני.

### אנימציות נוספות
- **לוגו הגלים** — מונפש תמיד (5 פסים כמו VU חי), drop-shadow סגול
- **פס VU קריטי** (90+ / אדום) — `glow` פועם, 2s ease-in-out
- **טעינת מסך** — כרטיסים נכנסים ברצף (stagger 60ms), לא בבת אחת
- **הכל** מכבד `prefers-reduced-motion: reduce` → מיידי, בלי תנועה

## 7. אייקונים — גאומטריים, קוויים, חדים

**אפס אימוג'י בכל המערכת.** SVG stroke-based, `stroke-width:1.5`, קצוות מעוגלים.
כל אייקון בקוביית גרדיאנט (34×34, `border-radius:11px`) עם צל צבעוני:

```
הפקות  → waveform (גלי סאונד)      גרדיאנט סגול
תוכניות → מצלמה                    גרדיאנט תכלת
חוזים  → מסמך עם קיפול פינה         גרדיאנט סגול-בהיר
משתמשים → אנשים                     גרדיאנט
רדאר   → מטרה/סריקה                 גרדיאנט
ארכיון → תיבה                       אפור (משני)
כספים  → שטר/מטבע                   גרדיאנט
```

קוביית אייקון:
```css
width:34px; height:34px; border-radius:11px;
background: linear-gradient(135deg, <accent>, <accent-dk>);
box-shadow: 0 4px 14px rgba(<accent>, 0.45);
```

## 8. כפתורים

- **ראשי (פעולה סופית/בלתי הפיכה):** מילוי גרדיאנט (סגול→סגול-כהה), טקסט לבן, צל צבעוני
- **רגיל (ניווט, פתיחה):** outline — גבול `rgba(255,255,255,0.15)`, רקע שקוף, hover מאיר
- **קריטי:** גבול/מילוי ורוד
- כולם: `border-radius:12px`, Space Grotesk 500, transition חלק

---

## 9. טבלאות (הפקות, כספים)

- רקע: `--bg-panel` עם border עדין
- כותרת: `--bg-elevated`, טקסט `--ink-dim` 10px letter-spacing
- שורה ב-hover: רקע `rgba(255,255,255,0.03)`
- מספרים: Space Grotesk, tabular, מיושר
- pills/תגיות: רקע שקוף-צבעוני (13% opacity) + טקסט בצבע מלא
- **RTL מלא** — יישור לימין, כיוון נכון

## 10. מצבים ריקים + כתיבה

- מסך ריק = הזמנה: "עדיין אין תוכניות. הוסף את הראשונה" — לא "No data"
- כתיבה בעברית, גוף פועל: "שולם" / "הוקפא" / "באיחור 34 יום"
- שגיאה = מה קרה + מה לעשות, בלי התנצלות
- **הערה עתידית:** מסכים ריקים יקבלו איור אווירה מ-Higgsfield אחרי בטא

## 11. יישום — סדר

יישם על כל המסכים הקיימים, בזה אחר זה:
1. **טוקנים גלובליים** — CSS variables + טעינת פונטים + שכבת גרדיאנט הרקע
2. **לוגו + ניווט עליון** (מופיע בכל מסך)
3. **הדשבורד (hub)** — הכי חשוב, זה שהמוקאפ הראה
4. **מסך תוכניות** + כרטיס תוכנית + שיוך יתומות
5. **מסך הפקות** (קנבן + לשונית עבודה) — המבחן על מסך צפוף
6. **רדאר** — מד ה-VU הוא הסיגנצ'ר, הזוהר קריטי כאן
7. **מסך הגדרות** + כל שאר המסכים
8. **מסך התחברות** — רושם ראשון

אחרי כל מסך — commit. הראה לי צילום של ה-hub קודם, לפני שממשיכים לשאר.

## 12. מה לא לעשות
- אין אימוג'י. אף פעם.
- אין flat design — הכל עומק, גרדיאנט, זכוכית.
- אין font ברירת מחדל של המערכת.
- אל תערבב חוב (ורוד/אדום) עם התחייבות (תכלת).
- אל תשבור RTL.
- אל תסטה מהטוקנים בלי לשאול את הבעלים.
